RDF Translator, powered by RDFLib 3.1.0

RDF Translator is a multi-format conversion tool for structured markup. It provides translations between data formats ranging from RDF/XML to RDFa or Microdata. The service allows for conversions triggered either by URI or by direct text input. Furthermore it comes with a straightforward REST API for developers.

  • URI
  • Input Field

Examples: RDFa - Microdata - RDF/XML - N3 - N-Triples - RDF/JSON

Rest API

This service provides an easily accessible API which allows for two variants:

  • "url", "if", and "of"
  • "url" and "of", while "if" is determined automatically

The API pattern is as follows (where the usage of "if" is more restrictive and optional):
http://rdf-translator.appspot.com/parse?url=<page_url>&if=<input_format>&of=<output_format>

if = [ rdfa | microdata | xml | n3 | nt | rdf-json | trix ]

of = rdfa | microdata | pretty-xml | xml | n3 | nt | rdf-json-pretty | rdf-json | trix

License and Source Code

The source code of this tool is available under a LPGL license from here.

This service is built on top of RDFLib 3.1.0, an RDF library which allows to parse and serialize between data formats. For the translation between microdata and the other file formats it is using Ed Summers' microdata plugin, for RDF/JSON the plugin as available in the RDFLib add-on package RDFExtras.
